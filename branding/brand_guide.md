# Clinic Brand Usage Guide

## Logo Overview

The Clinic logo is a clean, modern typographic mark that conveys professionalism and approachability. It uses Plus Jakarta Sans in bold weight with a distinctive accent on the "i" dot, which is styled as a small circle in the secondary pink color to add warmth and connect with the aesthetic clinic audience.

---

## Logo Variants

### 1. Logo Dark (Primary)
- **Usage:** Light backgrounds, main marketing materials
- **Text Color:** Slate 800 (#1E293B)
- **Accent (i dot):** Pink (#F472B6)
- **File:** `logo_dark.png`

### 2. Logo Light (Inverted)
- **Usage:** Dark backgrounds, hero sections, footer
- **Text Color:** White (#FFFFFF)
- **Accent (i dot):** Pink (#F472B6)
- **File:** `logo_light.png`

### 3. Logo Monochrome
- **Usage:** Single-color applications, watermarks, embossing
- **Color:** Single color (adapts to context)
- **File:** `logo_mono.png`

## Logo Construction

```
  ┌─────────────────────────────────────────┐
  │                                         │
  │   C l i n i c                           │
  │       ●                                 │
  │    (pink dot)                           │
  │                                         │
  └─────────────────────────────────────────┘
```

- Font: Plus Jakarta Sans Bold
- Letter-spacing: -0.02em (slightly tighter)
- The dot on the first "i" is replaced with a pink circle
- Circle size: 120% of default dot size
- Circle position: Centered above the stem

---

## Clear Space

Minimum clear space around the logo equals the height of the lowercase "c" in the wordmark.

```
  ┌───────────────────────────────────┐
  │     ↑                             │
  │     │ (clear space = "c" height)  │
  │     ↓                             │
  │  ←→ Clinic ←→                     │
  │     ↑                             │
  │     │                             │
  │     ↓                             │
  └───────────────────────────────────┘
```

---

## Minimum Size

- **Digital:** Minimum width of 80px
- **Print:** Minimum width of 20mm

---

## Logo Do's

- Use the logo on clean, uncluttered backgrounds
- Maintain proper clear space at all times
- Use approved color variants only
- Scale proportionally (lock aspect ratio)
- Use high-resolution assets for print
- Place on solid backgrounds or simple gradients

---

## Logo Don'ts

- Don't rotate or skew the logo
- Don't change the logo colors outside brand palette
- Don't add effects (shadows, glows, outlines)
- Don't stretch or distort proportions
- Don't place on busy or low-contrast backgrounds
- Don't rearrange or modify letterforms
- Don't add taglines directly attached to logo
- Don't use low-resolution versions for print
- Don't animate individual letters

---

## Color Usage Rules

### Primary Brand Color (#6366F1 Indigo)
- Use for: CTAs, buttons, links, active states, key highlights
- Minimum: Ensure 4.5:1 contrast ratio for text

### Secondary Brand Color (#F472B6 Pink)
- Use for: Accents, decorative elements, logo dot, feminine touch points
- Don't overuse: Max 20% of any design surface

### Success Green (#10B981)
- Reserved for: Positive states, completed actions, won deals
- Don't use for: Decorative purposes

### Warning Amber (#F59E0B)
- Reserved for: Alerts, pending items, attention-needed states
- Don't use for: Decorative purposes

### Error Red (#EF4444)
- Reserved for: Errors, destructive actions, lost leads
- Don't use for: Decorative purposes

---

## Typography Rules

### Headlines
- Font: Plus Jakarta Sans
- Weight: Bold (700) or Extra Bold (800)
- Case: Sentence case preferred

### Body Text
- Font: Inter
- Weight: Regular (400) or Medium (500)
- Size: 16px base for mobile readability

### Buttons & Labels
- Font: Inter
- Weight: Semi-Bold (600)
- Case: Sentence case for buttons

### Numbers & Data
- Font: Inter
- Use tabular figures for aligned numbers

---

## Icon Style Guidelines

### Recommended Library
**Heroicons** (by Tailwind CSS team)
- Style: Outline (24px) for navigation, Solid (20px) for inline
- Stroke: 1.5px for outline icons
- URL: https://heroicons.com

### Alternative Libraries
- **Lucide Icons** - Similar aesthetic, MIT license
- **Phosphor Icons** - Flexible weight options

### Icon Usage Rules
- Use consistent style throughout the app (don't mix outline/solid randomly)
- Navigation icons: Outline style
- Action buttons: Solid style
- Status indicators: Solid style
- Decorative/empty states: Outline style
- Size: 24px for navigation, 20px for inline, 16px for small indicators

### Icon Colors
- Default: Slate 500 (#64748B)
- Active/Selected: Primary (#6366F1)
- Success: Emerald (#10B981)
- Warning: Amber (#F59E0B)
- Error: Red (#EF4444)

## Application Examples

### Mobile App
- Logo in header: 32px height
- Navigation bar: 56px height
- Card padding: 16px
- Button height: 48px (touch-friendly)
- Border radius: 12px for cards, 8px for buttons

### Marketing
- Logo in hero: Prominent, centered or left-aligned
- Ample white space around all elements
- Pink accent used sparingly for highlights

### Social Media
- Profile picture: Simplified "C" mark with pink dot
- Consistent filter/overlay style for photos
- Brand colors in graphics and highlights
